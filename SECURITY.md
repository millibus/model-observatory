# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it privately.

**Please do not open a public GitHub issue for security concerns.**

Send a private vulnerability report via GitHub's [Private vulnerability reporting](https://github.com/millibus/model-observatory/security/advisories/new) feature, or contact the maintainers directly.

We will respond within 48 hours and work with you to understand and address the issue promptly.
